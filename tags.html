  • <a>

    For when you need a hyperlink.. for example click on each tag in the list.

  • <abbr>

    For when you need to make an abbreviation. (Hovering over it displays the full name) for example hover the cursor over CVJ 341.

  • <address>

    For when you would like to make an address significant.. for example the bottom of this page.

  • <blockquote>

    For when you would like to make a quote significant.. for example the indention of CVJ 341 at the top.

  • <cite>

    For when you would like to give citation to a specific item

  • <code>

    For when you would like to specify a segment of computer code (not ordinary text).

  • <del>

    For when you want a word/text to show it has been dleted. deleted.

  • <em>

    For when you want to have text in italics.

  • <h1-h6>

    For when you want to make text signicant as a heading.

  • <hr>

    For when you want to make a line seperating areas of the text/site.

  • <img>

    For when you would like an image incorporated into the site.

  • <ins>

    For when you would like to underline a unit of text. Show an "insert" as opposed to a "delete"

  • <kbd>

    A tag to distinguish text which is keyboard input from regular text.

  • <p>

    For when you need a paragraph, or ...

    a new line of text

  • <pre>

    For when you want to display text a predetermed,with line break and spaces, format.

  • <q>

    - Just a way to Make a quick quote.

  • <small>

    For when you would like some text to be smaller

  • <strong>

    For when you would like to BOLD some text.

  • <sub>

    For when you want subscripts.. 11012 = 1310

  • <sup>

    For when you want to use exponents. 22 = 4...

  • <var>

    For when you want a word to be a variable.
